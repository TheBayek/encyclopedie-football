# Site Web sur le Football

Ce site web statique contient 50 pages dédiées au football.

## Structure
- `index.html` : Page d'accueil
- `css/style.css` : Feuille de style
- `js/script.js` : Script JavaScript
- `pages/` : Dossier contenant les 50 pages (page1.html à page50.html)

## Lancement
Ouvrez `index.html` dans un navigateur web pour accéder au site.

## Développement
Le site est en HTML, CSS et JavaScript pur. Pas de compilation nécessaire.
